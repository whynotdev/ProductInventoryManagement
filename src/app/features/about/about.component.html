About This Application

This is a simple yet powerful inventory management system designed to help businesses efficiently manage their product inventory. Users can seamlessly add, edit, view, and delete products with ease. The system categorizes products, tracks stock levels, and provides real-time updates to ensure smooth inventory management.

The front-end is built using Angular, ensuring a dynamic and responsive user experience, while the back-end operates on a JSON server for data storage and API functionalities. This project serves as a great foundation for businesses looking to streamline their inventory tracking and management processes.
